# 양산외국인노동자의집 홈페이지 관리자 사용설명서

> IT 지식이 없는 분도 쉽게 따라할 수 있는 관리자 가이드입니다.

---

## STEP 1. 처음 시작하기

### 관리자 페이지 접속하기

1. 인터넷 브라우저(크롬, 엣지, 사파리 등)를 실행합니다.
2. 주소창에 관리자 페이지 주소를 입력합니다.
   - **실제 운영**: `https://admin.withmigrant.or.kr/login`
3. Enter 키를 누릅니다.

### 필요한 정보

- **아이디**: 관리자 계정 아이디
- **비밀번호**: 관리자 계정 비밀번호

> **💡 팁**: 관리자 계정 정보를 잊어버렸다면, 김찬에게 문의하세요(010-5060-1596)

---

## STEP 2. 로그인하기

### 로그인 절차

1. 관리자 페이지에 접속하면 로그인 화면이 나타납니다.
2. **아이디** 입력란에 관리자 아이디를 입력합니다.
3. **비밀번호** 입력란에 비밀번호를 입력합니다.
4. **로그인** 버튼을 클릭합니다.

### 로그아웃

1. 화면 오른쪽 하단의 **로그아웃** 메뉴를 클릭합니다.
2. 로그인 화면으로 돌아갑니다.

> **⚠️ 중요**: 작업이 끝나면 반드시 로그아웃하세요!

---

## STEP 3. 대시보드 살펴보기

로그인에 성공하면 대시보드 화면이 나타납니다.

### 대시보드에서 볼 수 있는 정보

#### 환영 메시지
- "**안녕하세요, (관리자명)님!**"
- "오늘도 좋은 하루 되세요."

#### 📊 통계 카드
- **공지사항**: 현재 등록된 공지사항 개수
- **활동후기**: 현재 등록된 활동후기 개수
- **소식지**: 현재 등록된 소식지 개수
- **대기 중 후원**: 아직 처리하지 않은 후원 신청 개수

#### 📝 최근 활동
- **최근 공지사항**: 최근에 작성된 공지사항 3개
- **최근 활동후기**: 최근에 작성된 활동후기 3개 (썸네일 포함)
- **최근 소식지**: 최근에 등록된 소식지 2개

### 왼쪽 메뉴 구성

#### 콘텐츠 관리
- **공지사항**: 홈페이지 공지사항을 작성하고 관리합니다
- **활동후기**: 활동 사진과 내용을 올립니다
  - **카테고리**: 활동후기 카테고리를 만들고 관리합니다
- **소식지**: Stibee로 제작한 소식지를 게시합니다

#### 사이트 설정
- **사이트 정보**: 단체명, 주소, 연락처 등 기본 정보를 관리합니다
- **메인 사진**: 메인 페이지에 표시될 주요 활동사진을 관리합니다
- **사업분야**: 주요 사업분야를 등록하고 수정합니다
- **연혁**: 단체의 역사를 연도별로 관리합니다
- **대중교통**: 사무실에 오시는 길 내용 중 버스 노선과 정류장 정보를 관리합니다
- **운영시간**: 사무실에 오시는 길 내용 중 사무실 운영 시간을 관리합니다

#### 함께하는 법
- **자원활동**: 자원봉사 모집 분야를 관리합니다
- **후원분야**: 후원금 모집 분야를 관리합니다
- **사용용도**: 후원금 사용 내역을 관리합니다

#### 후원 관리
- **후원신청**: 홈페이지를 통해 접수된 후원 신청을 확인합니다

#### 시스템
- **파일 관리**: 업로드된 파일을 확인하고 삭제합니다
- **빌드 히스토리**: 홈페이지 업데이트 기록을 확인합니다
- **관리자 계정**: 관리자 계정을 추가하거나 관리합니다 (슈퍼관리자만)

## STEP 4. 콘텐츠 관리하기

### 새로운 공지사항 작성하기

1. **콘텐츠 관리 > 공지사항** 메뉴를 클릭합니다.
2. **+ 새 공지사항** 버튼을 클릭합니다.
3. **제목**과 **내용**을 입력합니다.
4. 중요한 공지는 **"이 공지사항을 상단에 고정"** 을 체크합니다.
5. **저장** 버튼을 클릭하여 등록을 완료합니다.

### 활동후기 작성하기

1. **콘텐츠 관리 > 활동후기** 메뉴를 클릭합니다.
2. **+ 새 활동후기** 버튼을 클릭합니다.
3. **제목**을 입력하고 알맞은 **카테고리**를 선택합니다.
4. **썸네일 이미지**: 목록에 보일 대표 사진을 반드시 등록합니다.
5. **내용**을 작성한 후 **저장** 버튼을 클릭합니다.

### 소식지 등록하기

1. **콘텐츠 관리 > 소식지** 메뉴를 클릭합니다.
2. **+ 새 소식지** 버튼을 클릭합니다.
3. **Stibee에서 가져오기**: Stibee에서 제작한 소식지 URL을 입력하고 **불러오기** 버튼을 누릅니다.
4. **발행일**을 선택하고 **저장**을 누릅니다.

---

## STEP 5. 사이트 설정

### 사이트 기본 정보 수정

사이트 정보는 단체의 기본 정보(이름, 주소, 연락처 등)를 관리하는 메뉴입니다.

1. 왼쪽 메뉴에서 **사이트 설정 > 사이트 정보**를 클릭합니다.
2. 현재 등록된 정보가 표시됩니다.

#### 수정 가능한 항목

**기본 정보**
- **단체명**: 공식 단체명 (예: "(사)함께하는세상")
- **사이트명**: 홈페이지 이름 (예: "양산외국인노동자의집")
- **슬로건**: 단체 슬로건 (예: "더불어 사는 세상")
- **소개 문구**: 메인 페이지에 뜨는 활동사진과 함께 뜨는 메시지

**연락처 정보**
- **주소**: 사무실 주소
- **전화번호**: 대표 전화번호
- **팩스**: 팩스 번호
- **이메일**: 대표 이메일 주소

**후원 계좌**
- **은행명**: 후원금 입금 은행
- **계좌번호**: 후원금 입금 계좌번호
- **예금주**: 계좌 소유자 이름

### 메인 사진 관리

메인 페이지에 표시되는 대표 활동 사진을 관리할 수 있습니다.

#### 새로운 사진 추가하기

1. 왼쪽 메뉴에서 **사이트 설정 > 메인 사진**을 클릭합니다.
2. **+ 사진 추가** 버튼을 클릭합니다.
3. 정보를 입력합니다:
   - **이미지**: 사진 파일을 업로드합니다
   - **설명**: 사진에 대한 간단한 설명 (선택사항)
   - **촬영일**: 사진을 찍은 날짜 (선택사항)
4. **저장** 버튼을 클릭합니다.

#### 사진 순서 변경하기

1. 목록에서 사진 카드를 드래그하여 순서를 변경합니다.
2. 자동으로 저장됩니다.

#### 사진 활성화/비활성화

- 임시로 숨기고 싶은 사진의 **활성** 버튼을 누르면 **비활성**으로 변경되고,
- **비활성** 상태에서 다시 누르면 **활성**으로 변경됩니다.

### 사업분야 관리하기

사업분야는 단체가 수행하는 주요 사업을 소개하는 메뉴입니다.

#### 새로운 사업분야 추가하기

1. 왼쪽 메뉴에서 **사이트 설정 > 사업분야**를 클릭합니다.
2. **+ 사업분야 추가** 버튼을 클릭합니다.
3. 정보를 입력합니다:
   - **이름**: 사업분야 이름 (예: "인권상담")
   - **설명**: 간단한 설명 (1~2줄)
   - **상세 내용**: 자세한 설명 (사업분야 카드를 누르면 나오는 내용)
   - **사진**: 대표 이미지 업로드
4. **저장** 버튼을 클릭합니다.

#### 사업분야 순서 변경하기

1. 사업분야 목록에서 순서를 변경하고 싶은 항목을 찾습니다.
2. 드래그(마우스로 누른 채 이동)하여 순서를 바꿉니다.
3. 자동으로 저장됩니다.

### 연혁 설정하기

섹션(시기 구분)을 먼저 만들고, 그 안에 연도별 상세 항목을 추가합니다.
- **섹션**: 큰 시기 구분 (예: "시작", "성장기", "현재")
- **항목**: 각 연도별 세부 내용

#### 새로운 섹션 추가하기

1. 왼쪽 메뉴에서 **사이트 설정 > 연혁**을 클릭합니다.
2. **+ 섹션 추가** 버튼을 클릭합니다.
3. 정보를 입력합니다:
   - **부제목**: 섹션 이름 (예: "시작")
   - **요약 제목**: 요약 제목
   - **요약**: 이 시기에 대한 간단한 설명
4. **저장** 버튼을 클릭합니다.

#### 섹션에 연도별 항목 추가하기

1. 섹션 목록에서 항목을 추가할 섹션을 찾습니다.
2. **+ 항목 추가** 버튼을 클릭합니다.
3. 정보를 입력합니다:
   - **(연도)**: 4자리 연도(예:1997)를 씁니다. 이전 항목과 같은 연도인 경우 등 필요시 비워두세요.
   - **(내용)**: 해당 연도에 있었던 일
4. **저장** 버튼을 클릭합니다.

### 오시는 길 '대중교통' 설정

단체 소개 페이지의 "오시는 길" 내용 중 대중교통 안내 부분을 관리할 수 있습니다.

#### 사무실 주변 버스 정류장 추가하기

1. 왼쪽 메뉴에서 **사이트 설정 > 대중교통**을 클릭합니다.
2. 왼쪽 카드 **주변 정류장**을 보세요.
3. 정류장 이름을 입력하고 **추가** 버튼을 눌러 등록하세요.
   - 예: "양산시청 앞"

#### 사무실 주변을 오가는 버스 노선 추가하기

1. 왼쪽 메뉴에서 **사이트 설정 > 대중교통**을 클릭합니다.
2. 오른쪽 카드 **버스 노선**을 보세요.
3. 정보를 입력하고 **추가** 버튼을 눌러 등록하세요.
   - **버스 종류**: 일반/좌석/마을 중 선택
   - **버스 번호**: 예: "11", "26(출근)"

### 오시는 길 '운영시간' 설정

사무실 운영 시간과 휴무일을 관리할 수 있습니다.

#### 주요 사업 운영시간 추가하기

1. 왼쪽 메뉴에서 **사이트 설정 > 운영시간**을 클릭합니다.
2. 왼쪽 카드 **주요 사업 운영시간**을 보세요.
3. 정보를 입력하고 **추가** 버튼을 누르세요:
   - **사업명**: 예: "이주민 인권 상담"
   - **운영시간**: 요일별 정보를 작성하세요.
     ```
     월~목요일 : 10:00~18:00
     금~일요일 : 12:00~18:00
     ```

#### 사무실 정보 설정하기

1. 왼쪽 메뉴에서 **사이트 설정 > 운영시간**을 클릭합니다.
2. 오른쪽 카드 **사무실 정보**를 보세요.
3. **사무실 근무시간**을 작성하세요.
   - 예:
     ```
     월~목요일 : 10:00~18:00
     일요일 : 12:00~18:00
     ```
4. **휴무일**을 작성하세요.
   - 예: `금요일, 토요일은 쉽니다.`
5. **저장** 버튼을 누르세요.

## STEP 6. '함께하는 법' 설정

### '자원활동' 참여가능분야 설정

1. 왼쪽 메뉴에서 **함께하는 법 > 자원활동**을 클릭합니다.
2. **+ 분야 추가** 버튼을 클릭합니다.
3. 정보를 입력합니다:
   - **이름**: 분야 이름 (예: "교육활동지원")
   - **설명**: 상세 설명
4. **저장** 버튼을 클릭합니다.

### 후원을 모집하는 '후원분야' 설정

1. 왼쪽 메뉴에서 **함께하는 법 > 후원분야**를 클릭합니다.
2. **+ 분야 추가** 버튼을 클릭합니다.
3. 정보를 입력합니다:
   - **이름**: 분야 이름 (예: "물품후원")
   - **설명**: 설명(예: "사무실에 필요한 물건을 보내주실 수 있습니다")
4. **저장** 버튼을 클릭합니다.

### 후원금 '사용용도' 설정

1. 왼쪽 메뉴에서 **함께하는 법 > 사용용도**를 클릭합니다.
2. 사용용도 입력란에 내용을 작성하세요 (예: "산재 등 이주민 상담 지원에 쓰입니다")
3. **추가** 버튼을 눌러 등록하세요.

## STEP 7. 후원 관리

### 제출된 '후원신청'서 확인하기

1. 왼쪽 메뉴에서 **후원 관리 > 후원신청**을 클릭합니다.
2. 접수된 후원 신청이 목록으로 표시됩니다.
3. 필터 탭을 사용하여 상태별로 볼 수 있습니다:
   - **전체**: 모든 후원신청
   - **대기 중**: 아직 처리하지 않은 신청
   - **처리완료**: 처리가 끝난 신청

#### 후원 신청 상세 보기

1. 목록에서 확인하고 싶은 신청 건의 **상세** 버튼을 클릭합니다.
2. 다음 정보가 표시됩니다:
   - 신청자 이름, 생년월일
   - 주소, 전화번호
   - 후원 금액, 출금일
   - 계좌 정보
   - 신청 일시

#### 처리완료 표시하기

1. 후원 신청을 확인하고 처리했으면
2. **완료** 버튼을 클릭합니다.
3. 목록에서 "처리완료"으로 표시됩니다.

#### PDF로 출력하기

1. 후원 신청 목록에서 **PDF** 버튼을 클릭합니다.
2. 후원 신청서 PDF 화면이 열립니다.
3. 브라우저의 인쇄 기능을 사용하여 PDF로 저장하거나 인쇄할 수 있습니다.

## STEP 8. 시스템

### 현재 등록된 '파일 관리'

업로드된 이미지와 파일을 확인하고 삭제할 수 있습니다.
또한 전체 사용 용량과 남은 용량을 볼 수 있습니다.

#### 파일 목록 보기

1. 왼쪽 메뉴에서 **시스템 > 파일 관리**를 클릭합니다.
2. 업로드된 모든 파일이 표시됩니다.
3. 다음 정보를 확인할 수 있습니다:
   - 파일명
   - 파일 종류 (이미지, PDF 등)
   - 파일 크기
   - 업로드 날짜

#### 파일 삭제하기

1. 삭제하고 싶은 파일을 찾습니다.
2. **삭제** 버튼을 클릭합니다.
3. 확인 메시지에서 **확인**을 클릭합니다.

> **⚠️ 주의사항**:
> - 삭제한 파일은 복구할 수 없습니다
> - 게시물에서 사용 중인 파일을 삭제하면 이미지가 깨질 수 있습니다

#### 저장 공간 확인하기

- 화면 상단에 전체 저장 공간 사용량이 표시됩니다
- **전체 한도**: 10GB
- 현재 사용량과 남은 용량을 확인할 수 있습니다

### '관리자 계정' 생성/삭제/권한부여 하기 (슈퍼관리자 전용)

새로운 관리자를 추가하거나 권한을 변경할 수 있습니다.

> **⚠️ 주의**: 이 기능은 슈퍼관리자만 사용할 수 있습니다.

#### 권한 구조
- **관리자**: 관리자 추가/생성/정보 변경(비밀번호 포함) 불가
- **슈퍼관리자**: 관리자 추가/생성/정보 변경(비밀번호 포함) 가능

#### 새 관리자 추가하기

1. 왼쪽 메뉴에서 **시스템 > 관리자 계정**을 클릭합니다.
2. **새 관리자** 버튼을 클릭합니다.
3. 정보를 입력합니다:
   - **아이디**
   - **비밀번호**
   - **이름**
   - **권한** (관리자 또는 슈퍼관리자)
4. **저장** 버튼을 클릭합니다.

#### 관리자 권한 변경하기

1. 관리자 목록에서 수정할 관리자를 찾습니다.
2. **수정** 버튼을 클릭합니다.
3. 권한을 변경합니다.
4. **저장** 버튼을 클릭합니다.

#### 관리자 계정 삭제하기

1. 관리자 목록에서 삭제할 관리자 계정을 찾습니다.
2. **삭제** 버튼을 클릭합니다.
3. 해당 관리자는 더 이상 로그인할 수 없습니다.

## 자주 묻는 질문 (FAQ)

### Q1. 로그인이 안 됩니다.
**A**: 다음을 확인하세요:
- 아이디와 비밀번호를 정확히 입력했나요?
- 대문자/소문자를 구분해서 입력했나요?
- 계정이 비활성화되었나요? (시스템 담당자에게 문의)

### Q2. 이미지가 업로드되지 않습니다.
**A**: 다음을 확인하세요:
- 파일 크기가 10MB 이하인가요?
- 파일 형식이 JPG, PNG, GIF, WEBP 중 하나인가요?
- 저장 공간이 부족하지 않나요? (파일 관리에서 확인)

### Q3. 작성한 내용이 홈페이지에 바로 표시되지 않습니다.
**A**:
- 시스템이 자동으로 홈페이지를 업데이트합니다 (1~2분 소요)
- 잠시 기다린 후 홈페이지를 새로고침(F5)하세요
- 계속 표시되지 않으면 시스템 담당자에게 문의하세요

### Q4. 실수로 삭제했어요. 복구할 수 있나요?
**A**:
- 죄송하지만 삭제된 내용은 복구할 수 없습니다
- 중요한 내용은 삭제 전에 꼭 확인하세요

### Q5. 카테고리를 추가하고 싶어요.
**A**:
1. 왼쪽 메뉴에서 **콘텐츠 관리 > 카테고리**를 클릭합니다
2. **+ 카테고리 추가** 버튼을 클릭합니다
3. 이름과 색상을 선택하고 저장합니다
